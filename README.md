# Smart Food Donation System

A web-based platform that connects food donors with receivers to reduce food waste and help needy people.

## 🚀 Features

- User Registration & Login
- Donor Dashboard
- Receiver Dashboard
- Food Upload with Image
- Real-time Chat between Donor and Receiver
- Secure Authentication System

## 🛠 Technologies Used

- Python
- Flask
- HTML
- CSS
- SQLite (if used)

## 📂 Project Structure

Smart_food_Donation_system/
│── app.py  
│── static/  
│── templates/  

## ▶ How to Run

1. Install dependencies:
pip install flask

2. Run the application:
  app.py 
3. Open browser


## 🎯 Objective

To reduce food wastage and create a bridge between donors and receivers using a simple web platform.
